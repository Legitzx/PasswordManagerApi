# PasswordManagerApi

The PasswordManagerApi is a REST Api that handles CRUD operations for the [PasswordManager](https://github.com/Legitzx/PasswordManager).

## Functionality


**Method: [POST]** 
```
/register
/login
```
**Method: [PUT]**
```
/update
```
**Method: [GET]**
```
/get
```
